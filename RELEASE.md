# 发布流程

## v1.2.12 搜索与列表低分配优化

- `v1.2.12` 继续围绕播放器启动、搜索输入、左侧本地曲库面板和 3D 歌单架刷新做低风险性能优化，不改变视觉、播放和歌单架交互逻辑。
- 本地搜索索引文本拼接改为直接拼接字段，搜索和索引预热时不再为每首歌创建字段数组和 `filter` 回调。
- 左侧本地曲库面板可见卡片改为单次循环生成 HTML，滚动加载更多时减少 `slice/map/join` 中间数组。
- 3D 歌单架非队列数据签名改为直接采样头尾项，减少 rebuild 判断时的临时数组和回调分配。
- 本次发布上传完整安装包、blockmap、`latest.yml`、Portable ZIP、SHA256 校验文件和 `1.2.11 -> 1.2.12` 快速补丁。

## v1.2.11 本地曲库低分配性能优化

- `v1.2.11` 继续围绕本地大曲库恢复、导入和后台封面/歌词补齐做低风险性能优化，不改变左侧歌单显示/隐藏、固定按钮和 3D 歌单架“自动隐藏 / 常驻”逻辑。
- 本地资产缓存补水改为按范围读取，分块阶段不再反复 `slice` 生成临时数组，启动恢复和大曲库导入后更少主线程分配。
- 后台资源预载候选、播放队列位置映射和排序队列改为单次循环生成，并复用同一轮候选结果，减少封面/歌词补齐启动前的重复扫描。
- 搜索结果、队列和歌单详情等列表入场动画只收集实际需要动画的前几项，避免完整 NodeList 转数组后再截断。
- 本次发布上传完整安装包、blockmap、`latest.yml`、Portable ZIP、SHA256 校验文件和 `1.2.10 -> 1.2.11` 快速补丁。
- Release 标题建议使用 `Mineradio v1.2.11 本地曲库低分配性能优化版`。
