# 发布流程

## v1.2.26 迷你播放器稳定性修复
- `v1.2.26` 修复迷你播放器运行一段时间后被其他窗口覆盖、再次关闭主界面仍无法重新显示的问题。
- 迷你窗口显示时重新确认原生置顶状态；健康窗口只刷新 Z 序，被系统隐藏、最小化或显示器变化后自动拉回，锁屏/休眠期间暂停恢复定时器。
- 用户拖动位置会持久化并在重启后恢复，程序校正与用户移动分离，同一坐标不重复写盘。
- 播放状态同步不再重复解析歌曲和封面；IPC 返回失败会按字段失效缓存并重发，空队列和封面加载失败状态可正确恢复。
- 渲染进程首次崩溃优先重载，成功加载前再次崩溃会升级为窗口重建；旧窗口回调不会误清理新实例。
- 优化无歌词占位检测，减少 LRC/YRC/自定义歌词过滤时的临时字符串分配。
- 本次发布继续只上传安装器相关资产：安装包、blockmap、`latest.yml`、SHA256 校验文件和 `1.2.25 -> 1.2.26` 快速补丁；Portable ZIP 跳过。
- Release 标题使用 `Mineradio v1.2.26 迷你播放器稳定性修复版`。
